# Sales_Analysis_EDA_Project


Project Overview:
This project analyzes sales data to understand customer purchasing behavior and identify trends that can help businesses improve sales strategies. The analysis is performed using Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn.

Objectives:
Clean and preprocess the sales dataset.
Analyze customer demographics such as gender, age group, marital status, occupation, and state.
Identify the categories and products generating the highest sales.
Visualize key insights using charts and graphs.

Technologies Used:
Python
Jupyter Notebook
Pandas
NumPy
Matplotlib
Seaborn

Dataset:
The dataset contains information about customer purchases during the Diwali festival season, including:
User ID
Customer Name
Product ID
Gender
Age Group
Marital Status
State
Occupation
Product Category
Orders
Amount

Data Preprocessing:
Removed unnecessary columns.
Checked for missing values.
Handled null values where necessary.
Converted data into appropriate formats for analysis.
Analysis Performed
Gender-wise purchasing behavior.
Age group analysis.
Marital status analysis.
State-wise sales analysis.
Occupation-wise sales analysis.
Product category analysis.
Top-selling products analysis.


Key Insights:
Female customers contributed significantly to total sales.
Customers aged 26–35 years made the highest number of purchases.
Married customers showed higher purchasing activity.
Certain states contributed more revenue than others.
IT, Healthcare, and Aviation professionals were among the major customer groups.


Conclusion:
The analysis provides valuable insights into customer behavior during the Diwali season. Businesses can use these findings to target specific customer segments, improve marketing campaigns, and increase future sales.

Future Scope:
Build predictive models for sales forecasting.
Perform customer segmentation using machine learning.
Develop an interactive dashboard using Tableau or Power BI.
